# Cosmic Shore — Landing Page

The official landing page for **Cosmic Shore**, the party game for pilots by [Froglet Games](https://frogletgames.itch.io/cosmic-shore).

## Quick Start

This is a static site — no build step required. Just open `index.html` in a browser, or deploy to any static host.

## Hosting

### GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main** branch, **/ (root)**
4. Your site will be live at `https://YOUR_USERNAME.github.io/cosmic-shore-site/`

### Netlify
Drag and drop this entire folder onto [Netlify Drop](https://app.netlify.com/drop).

### Any Static Host
Upload the entire folder (preserving the directory structure) to any web server.
